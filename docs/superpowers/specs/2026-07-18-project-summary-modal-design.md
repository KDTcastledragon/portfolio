# 프로젝트 요약 모달 설계

## 목표

좌측 상단 페이지네이션 오른쪽 100px 지점에 `[ 6개 프로젝트 참여. (개인4 + 팀2) ]` 버튼을 배치하고, 클릭 시 여섯 프로젝트의 유형과 한 줄 설명을 보여준다.

## 구성

- 전용 네이티브 `<dialog id="projectSummaryDialog">`를 사용한다.
- 기존 `data-open`, `data-close`, 바깥 영역 클릭 처리 코드를 재사용한다.
- 네이티브 `<dialog>`의 기본 동작으로 `Esc` 닫기를 지원한다.
- 개인 프로젝트는 CastleChat, 여우 쇼핑몰, Respla 스터디카페, IP LIST다.
- 팀 프로젝트는 HPDFS, 가스 누출 방지 시뮬레이션이다.
- 데스크톱에서는 요청 문구 전체를 표시하고 모바일에서는 `프로젝트 6개`로 축약한다.

## 범위

- `index.html`과 `index.css`만 수정한다.
- 기존 프로젝트 카드, 상세 모달과 페이지 이동 로직은 변경하지 않는다.
