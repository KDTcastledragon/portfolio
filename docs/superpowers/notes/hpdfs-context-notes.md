# HPDFS 포트폴리오 작업 기록

## 2026-07-18

- 대상은 두 번째 페이지에서 프로젝트 번호 `2`로 표시된 기존 SMARTCTL 카드다.
- 프로젝트 번호 `3` 가스 누출 방지 시뮬레이션은 변경하지 않는다.
- 프로젝트 전체 아키텍처와 기술 스택에는 RandomForest 및 ML 구성요소를 표시할 수 있다.
- 사용자는 데이터 분석과 모델 학습을 담당하지 않았다.
- 개인 기여와 트러블 슈팅은 FastAPI API 설계와 PostgreSQL DB 설계로 제한한다.
- 실제 코드에서 확인한 DB 구조는 `diagnosis_log` 누적 이력과 `disks` 최신 상태 분리다.
- 실제 코드에서 확인한 API는 진단 요청, 전체 목록, 상세 이력, 조치 상태 변경, 데모 데이터 관리다.
- 라이브 AWS 배포는 종료된 상태이므로 배포 링크를 면접용으로 과장하지 않는다.
- 기존 포트폴리오 작업 트리가 더러운 상태이므로 관련 없는 변경을 보존한다.
- 프로젝트 번호 2의 배포 버튼은 실제 공개 저장소를 가리키는 GitHub 링크로 교체했다.
- 820px 이하에서는 프로젝트 목록을 1열로, 720px 이하에서는 카드 내부를 1열로 전환해 모바일 가독성을 확보했다.
- Node 기반 콘텐츠 및 JavaScript 검사와 Edge 헤드리스 데스크톱 및 500px 렌더링 검사를 통과했다.
