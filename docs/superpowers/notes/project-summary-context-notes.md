# 프로젝트 요약 모달 작업 기록

## 2026-07-18

- 전용 `<dialog>`를 사용한다.
- 기존 공통 모달 이벤트를 재사용해 JavaScript 추가를 최소화한다.
- CastleChat, 여우 쇼핑몰, Respla 스터디카페, IP LIST는 개인 프로젝트로 표시한다.
- HPDFS와 가스 누출 방지 시뮬레이션은 팀 프로젝트로 표시한다.
- 기존 작업 트리가 더러운 상태이므로 관련 없는 변경과 커밋은 건드리지 않는다.
- 페이지네이션 전체 폭 106px 뒤에 100px을 더한 206px 오프셋으로 데스크톱 위치를 계산했다.
- 980px 이하에서는 버튼을 우측으로 옮기고 `프로젝트 6개`로 축약한다.
- Edge 헤드리스 1440×900 및 500×844 렌더링에서 버튼과 모달을 확인했다.
